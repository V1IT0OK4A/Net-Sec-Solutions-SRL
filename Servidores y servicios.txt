## 🖥️ Servidor 1: DHCP + DNS (IP: 10.17.0.44)

### 1. Configuración de red (Netplan)

*Archivo:* /etc/netplan/00-installer-config.yaml

yaml
network:
  version: 2
  ethernets:
    ens3:
      addresses:
        - 10.17.0.44/28
      routes:
        - to: 10.16.0.0/16
          via: 10.17.0.33
        - to: 10.18.0.0/16
          via: 10.17.0.33
        - to: 10.17.0.0/24
          via: 10.17.0.33
        - to: default
          via: 10.17.0.33
          metric: 100
      nameservers:
        addresses:
          - 8.8.8.8
          - 8.8.4.4
    ens4:
      dhcp4: true
      dhcp4-overrides:
        route-metric: 50
        use-dns: false
      routes:
        - to: default
          via: 192.168.10.2
          metric: 50


*Aplicar:* sudo netplan apply

### 3. Servicio de Correo (Postfix + Dovecot) – Resumen de archivos clave

*Usuario virtual:*  
bash
sudo groupadd -g 5000 vmail
sudo useradd -r -u 5000 -g vmail -d /var/mail/vhosts -s /usr/sbin/nologin vmail
sudo mkdir -p /var/mail/vhosts/netsec.com.do
sudo chown -R vmail:vmail /var/mail/vhosts


*Archivo:* /etc/postfix/main.cf (agregar al final)


myhostname = mail.netsec.com.do
mydomain = netsec.com.do
myorigin = $mydomain
inet_interfaces = all
mynetworks = 127.0.0.0/8, 10.0.0.0/8

virtual_mailbox_domains = /etc/postfix/virtual_domains
virtual_mailbox_base = /var/mail/vhosts
virtual_mailbox_maps = hash:/etc/postfix/virtual_mailboxes
virtual_alias_maps = hash:/etc/postfix/virtual_aliases
virtual_minimum_uid = 100
virtual_uid_maps = static:5000
virtual_gid_maps = static:5000

smtpd_sasl_auth_enable = yes
smtpd_sasl_type = dovecot
smtpd_sasl_path = private/auth
smtpd_sasl_security_options = noanonymous
smtpd_tls_auth_only = yes
smtpd_recipient_restrictions = permit_mynetworks, permit_sasl_authenticated, reject_unauth_destination


*Archivos de mapeo Postfix:*
- /etc/postfix/virtual_domains con el contenido: netsec.com.do  
- /etc/postfix/virtual_mailboxes (vacío)  
- /etc/postfix/virtual_aliases (vacío)  

Ejecutar: sudo postmap /etc/postfix/virtual_domains

*Archivo:* /etc/dovecot/dovecot.conf → protocols = imap pop3 lmtp

*Archivo:* /etc/dovecot/conf.d/10-auth.conf → comentar !include auth-system.conf.ext y descomentar !include auth-passwdfile.conf.ext

*Archivo:* /etc/dovecot/conf.d/10-mail.conf  

mail_location = maildir:/var/mail/vhosts/%d/%n
mail_uid = vmail
mail_gid = vmail


*Archivo:* /etc/dovecot/conf.d/auth-passwdfile.conf.ext  

passdb {
  driver = passwd-file
  args = scheme=PLAIN username_format=%u /etc/dovecot/users
}
userdb {
  driver = static
  args = uid=vmail gid=vmail home=/var/mail/vhosts/%d/%n
}
